Component and Architecture Overview

Authentication

The prototype provides a foundational architecture for the registration and authentication of users.

Profile Management

Once authorized, the prototype provides profile management for all users.

Comments

Through inner joining data structures, the prototype is able to provide user focused CRUD operations for each object.

Styles

Through importing a variable driven SCSS file, the prototype's RBA and font values are able to be modified across the site.

MySQL CRUD

Driven off of a PHP backend, Angular's HTTP requests are able to effectively create, update, and delete the prototype's content.

Object Detail

The utilization of nested components within the code architecture provides each object to be viewed individually.

Cart

The prototype's cart allows the user to store, remove, and manage the quantity of the objects listed.

Search

The application's MySQL driven search function allows the user to search for any property of an object.

Bootstrap Tabs

TypeScript and Bootstrap provide the ability to iterate through objects providing an architecture that is easily modifiable.
